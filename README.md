# 📌 Descrição / Description

🇧🇷 **Português (BR)**  
Este repositório tem como objetivo centralizar e organizar os códigos que desenvolvi ao longo dos meus estudos da linguagem. Ele funciona como um espaço de prática, onde registro listas de exercícios, pequenos projetos e experimentos feitos para fixar conceitos fundamentais e explorar recursos mais avançados.  

A ideia é manter um histórico de evolução, permitindo acompanhar meu progresso, revisitar soluções anteriores e, ao mesmo tempo, servir como referência para consultas futuras. Além disso, este repositório pode ser útil para compartilhar aprendizados com outras pessoas que também estejam estudando a linguagem, criando um ambiente aberto para troca de conhecimento e boas práticas de programação. ✨💻  

---

🇺🇸 **English (EN)**  
This repository aims to centralize and organize the code I have developed throughout my studies of the language. It serves as a practice space, where I record exercise lists, small projects, and experiments created to reinforce fundamental concepts and explore more advanced features.  

The idea is to maintain a history of progress, allowing me to track my development, revisit previous solutions, and at the same time serve as a reference for future consultations. In addition, this repository can be useful for sharing knowledge with others who are also studying the language, fostering an open environment for exchanging ideas and programming best practices. ✨💻
